## Описание программы

**Weather Diary** — это консольное приложение для ведения дневника погоды. Позволяет добавлять записи о температуре, осадках, фильтровать их по дате, просматривать статистику и сохранять данные в JSON-файл.

Программа решает проблему хаотичного хранения погодных наблюдений, предоставляя структурированный и удобный способ записи и анализа погодных данных.

---

## Возможности программы

- ✅ Добавление записей (дата, температура, описание, наличие осадков)
- ✅ Просмотр всех записей в табличном виде
- ✅ Фильтрация записей по конкретной дате
- ✅ Фильтрация записей с температурой выше 20°C
- ✅ Статистика (средняя, максимальная, минимальная температура)
- ✅ Сохранение данных в JSON-файл
- ✅ Автоматическая загрузка данных при запуске

---

## Требования для запуска

Для работы программы необходимо установить:

- **Python 3.10** или выше
- Дополнительные библиотеки не требуются (используются только стандартные модули: `json`, `os`, `datetime`)
